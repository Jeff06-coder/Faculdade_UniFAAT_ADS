# Diferencia de Postgre para MongoDB
O Mongo é mais facil de se criar, e de se consultar.Muito pratico na inserção de dados e para consulta-los.
Já o Postgre, ele é mais complexo, segue mais "regrinhas" e sua inserção de dados para autos niveis assim é mais dificil, sua pesquisa também.Seus metodos de criar tabelas e pesquisas.
